# AIRag Application

## Overview
AIRag is a comprehensive application designed for effective financial analysis using advanced algorithms and LLM integration.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/king4arabs/AIRag.git
   ```
2. Navigate to the project directory:
   ```bash
   cd AIRag
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To start the application, run:
```bash
./run.sh
```

## Structure
- **src/**: Contains the main application logic.
- **backend/**: Contains the FastAPI application.
- **frontend/**: Contains the frontend files.
